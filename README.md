## About This Repository

This repository contains all weekly assignment notebooks completed as part of the Celebal Excellence Internship (CEI) program in the Data Science domain. Each week covers a structured set of topics progressing from foundational ML concepts to deep learning and time series forecasting.

## Weekly Breakdown

| Week | Topic | Dataset | Key Techniques |

| [Week 1](#week-1--ml-foundations) | ML Foundations | Custom employee data | Python, NumPy, Pandas, Statistics, Probability |

## Week 1 — ML Foundations

**Notebook:** `Week 1/week1_saksham_saraogi_jecrc.ipynb`

Covers the core Python and mathematical toolkit required for machine learning — from data structures and array operations to statistical inference and probability distributions.

Topics: Python fundamentals · NumPy · Pandas · Linear Algebra · Descriptive Statistics · Probability Theory · Distribution Visualization

[→ View Week 1 README](Week%201/README.md)

## Week 2 — Supervised Learning & Time Series

**Notebook:** `Week 2/week2_saksham_saraogi_jecrc.ipynb`  
**Dataset:** `Week 2/tesla_deliveries_dataset_2015_2025.csv`

An end-to-end ML pipeline on Tesla EV delivery data covering everything from raw data cleaning to hyperparameter-tuned regression models and SARIMA time series forecasting.

Topics: Data Cleaning · EDA · Encoding · Feature Scaling · Feature Engineering · Leakage Prevention · Sklearn Pipelines · Chronological Split · Linear/Ridge/Lasso Regression · Cross Validation · GridSearchCV · Stationarity (ADF/KPSS) · SARIMA

[→ View Week 2 README](Week%202/README.md)

---

## Week 3 — Unsupervised Learning

**Notebook:** `Week 3/week3_saksham_saraogi_jecrc.ipynb`  
**Dataset:** `Week 3/Country-data.csv`

Clusters 167 countries by socio-economic indicators using K-Means and DBSCAN, with PCA-based 2D visualization and silhouette score evaluation.

Topics: K-Means Clustering · Elbow Method · Silhouette Score · DBSCAN · PCA · StandardScaler

[→ View Week 3 README](Week%203/README.md)

---

## Week 4 — Computer Vision: ANN vs CNN

**Notebook:** `Week 4/week4_saksham_saraogi_jecrc.ipynb`

Builds and benchmarks multiple neural network architectures on CIFAR-10 — from a flat dense ANN baseline to a scaled-up CNN with data augmentation and early stopping.

Topics: ANN · CNN · Conv2D · BatchNorm · MaxPooling · Data Augmentation · EarlyStopping · CIFAR-10

[→ View Week 4 README](Week%204/README.md)

---

## Week 5 — Deep Learning: Text Generation

**Notebook:** `Week 5/week5_saksham_saraogi_jecrc.ipynb.ipynb`

Trains and compares three recurrent architectures (RNN, LSTM, GRU) on a custom text corpus for next-word prediction and coherent text generation.

Topics: Vanilla RNN · LSTM · GRU · Tokenization · N-gram Sequences · Embedding Layers · Text Generation
